# hipo_asmm_server

# node_sysfs_gpio

A Web GUI server for asmm device driver

## Installation

Install via git clone

```bash
$ git clone https://github.com/david6miji/hipo_asmm_server.git
$ cd hipo_asmm_server/app
$ npm install
```

## Run

run server

```bash
$ DEBUG=app:* npm start
```
